🎸 Groupie Tracker

Explore music bands and artists with live data, beautiful UI, and interactive features — powered by Go and the Groupie API.

Key Features

  • 🔍 Search artists by name, creation year, or first album
  • 🧑‍🎤 View band members and artist images
  • 📅 Explore concert dates and locations
  • 📱 Fully responsive and animated interface
  • ⚡ Data fetched live from Groupie Tracker API

Project Structure

. ├── main.go # Entry point & routing ├── go.mod # Module config ├── /internal │ ├── fetch.go # API data functions │ ├── struct.go # Data models ├── /templates │ ├── index.html # Homepage with artist cards │ ├── details.html # Artist details page ├── /static │ ├── style1.css # Homepage styles │ └── details.css # Details page styles

How It Works

  1. Homepage fetches all artists and displays cards
  2. Clicking a card redirects to detailed view
  3. Details include members, first album, and locations
  4. Go templates render HTML using API data

API Integration

  • /api/artists → All artists data
  • /api/relation/{id} → Tour dates and locations
  • Data fetched with net/http and encoding/json

How to Run

  1. Make sure you have Go installed
  2. Clone the project and run:
    go run main.go
  3. Visit http://localhost:8080 in your browser

Notes

  • Search engine is simple but expandable (search.go)
  • Some links and JSON displays are raw (for demo)
  • ID 21 excluded manually for testing
