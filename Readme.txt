# Student Profile Dashboard

This is a responsive personal portfolio page created using the iPortfolio Bootstrap template. It is designed to showcase a student's profile, including their education, experience, projects, and contact information.

## 🚀 Features

- Smooth scroll navigation menu
- Sections: Home, About, Education, Experience, Projects, Contact
- Responsive design (mobile & desktop)
- Bootstrap Icons for social media and navigation
- Easy to customize

## 🛠️ Built With

- HTML5
- CSS3
- Bootstrap 5.3
- JavaScript
- AOS (Animate on Scroll)
- Bootstrap Icons
