🧵 Real-Time Forum

A live chat & post platform built with Go, WebSocket, and HTML – all in real-time.

Features

  • 🧑‍💼 User registration and login
  • ✍️ Create and browse public posts
  • 💬 Commenting system for discussions
  • 🔒 Private messaging between users
  • 🟢 Live online user tracking
  • ✉️ Typing indicator notifications
  • 🔐 Session management via UUID
  • 🧠 View past messages sorted by recency

Requirements

  • Go 1.20 or higher
  • Node.js (optional for development)
  • SQLite
  • A modern browser with WebSocket support

Project Structure

. ├── main.go # HTTP server entry point ├── index.html # Frontend UI ├── app.js # JavaScript WebSocket logic ├── databace.go # Database handling ├── ws.go # WebSocket events ├── session.go # Online sessions ├── struct.go # Structs for users, messages, etc. ├── /public/ # Static CSS/JS files └── /data.db # SQLite database

How to Run

  1. Ensure Go and SQLite are installed
  2. Run the backend server:
    go run main.go
  3. Open in browser:
    http://localhost:8080

Test Endpoint

Visit this link to check the server response:

http://localhost:8080/exemple

You should see: Exemple endpoint is working!

Notes

  • All data is stored in data.db
  • Sessions reset on server restart
  • WebSocket endpoint: ws://localhost:8080/ws

Database Tables

  • users – user information
  • posts – forum posts
  • comments – post comments
  • private_messages – private chats
  • notifications – alerts (WIP)

Developer Note

This project showcases how real-time interactivity can be achieved using only Go, WebSocket, and frontend technologies – no frameworks, just speed and logic.

“Build real-time systems with real code, not magic.”
