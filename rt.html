How to Run

Run the raytracer in release mode for maximum performance:

cargo run --release <object_name>

Example:

cargo run --release cube

This command renders a cube based on the camera setup in camera.rs.

Camera Control

Modify camera position and direction in camera.rs:

let look_from = Vec3::new(...); // Camera position let look_at = Vec3::new(...); // Where the camera looks

Adjust these for front, top, or side views in the Camera::new() function.

Brightness & Lighting

To control scene brightness, update this line in main.rs:

color = (color / (samples_per_pixel as f32)) * 1.5;

Increase 1.5 to brighten, decrease it for darker results.

Output

After rendering, the final image is saved as:

output.png

You can open it with any image viewer (e.g. Windows Photos, GIMP, etc.).
